# Customer Behavior Analysis

## Overview
This project analyzes customer shopping behavior using SQL, Python, and Power BI.
The goal is to understand purchasing patterns, customer segments, and business insights from the data.

## Files in this Repository

- customer_shopping_behavior.csv  
  Raw dataset containing customer transactions

- customer_behavior.ipynb  
  Data analysis and preprocessing using Python

- customer_behavior_sql_queries.sql  
  SQL queries used for extracting insights

- customer_behavior_dashboard.pbix  
  Power BI dashboard for visualization

## Key Insights

- Revenue comparison between male and female customers  
- Impact of discounts on purchase behavior  
- Top-rated and most purchased products  
- Customer segmentation (New, Returning, Loyal)  
- Subscription vs non-subscription spending patterns  
- Shipping type impact on purchase amount  
- Age group contribution to revenue  

## Tools and Technologies

- Python (Pandas, Data Analysis)
- SQL (PostgreSQL / MySQL)
- Power BI (Dashboard and Visualization)

## Project Workflow

1. Data Collection  
2. Data Cleaning and Processing (Python)  
3. Data Analysis (SQL Queries)  
4. Data Visualization (Power BI Dashboard)  
5. Insights and Conclusions  

## How to Use

1. Download the dataset  
2. Run SQL queries in your database  
3. Open the notebook for analysis  
4. Load the Power BI file to view dashboard  

## Conclusion

This project helps in understanding customer behavior, improving marketing strategies, and making data-driven business decisions.
